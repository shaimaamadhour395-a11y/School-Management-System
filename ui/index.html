About the Project

This system integrates multiple database modules including Student Management, Teacher & Subject Management, and Class, Attendance & Grades. All modules were developed separately using Git feature branches and then merged into one unified system.

Student Module

Manage students and class assignments

Teacher & Subject Module

Manage teachers and subjects

Attendance & Grades

Track attendance and record grades
